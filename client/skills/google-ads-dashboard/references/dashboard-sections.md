# Dashboard Sections

## 1. Executive Summary Cards
Top row of KPI cards with current value and period-over-period delta:
- Total Spend (₹)
- Impressions
- Clicks
- CTR (%)
- Avg CPC (₹)
- Conversions
- Conversion Rate (%)
- CPA (₹)

Each card shows: metric name, value, trend arrow (▲/▼), % change vs previous period.

## 2. What Changed (Period Comparison)
Side-by-side comparison bar or column chart:
- Current period vs Previous period for: Spend, CPA, CPC, Conversions, CTR
- Color-coded: green = improvement, red = deterioration

## 3. Daily Trend Charts
Multi-line/area charts, one per row or tabbed:
- Spend over time
- Clicks & Impressions over time
- CPA over time
- Conversions over time

## 4. Growth Diagnosis
Narrative insight cards generated by the AI:
- Where budget is going (pie/donut chart)
- Where conversions are coming from
- What is wasting spend
- What deserves scaling
- What needs restructuring

## 5. Campaign Performance Table
Sortable, filterable table with columns:
- Campaign name
- Status
- Spend (₹)
- Clicks
- Impressions
- CTR
- Avg CPC (₹)
- Conversions
- CVR (%)
- CPA (₹)
- Impression Share
- Lost IS (Budget)
- Lost IS (Rank)
- AI Recommendation Label

## 6. Ad Group Performance Table
Same structure as campaign table, grouped under campaigns.

## 7. Keyword Performance Table
Sortable table:
- Keyword text
- Match type
- Status
- Campaign
- Ad Group
- Spend (₹)
- Clicks
- Impressions
- CTR
- Avg CPC (₹)
- Conversions
- CVR (%)
- CPA (₹)
- AI Label (Scale / Pause / Watch)

## 8. Search Term Mining Table
- Search term
- Status (Added/Excluded/None)
- Campaign
- Ad Group
- Spend (₹)
- Clicks
- Impressions
- CTR
- Conversions
- CPA (₹)
- AI Label (Negative candidate / Promote candidate / Watch)

## 9. AI Proposal Inbox
Card-based layout:
- Each card shows: type, priority badge, risk badge, summary, evidence metrics,
  suggested action, human status
- Action buttons: Accept / Reject / Watchlist / Needs More Data
- Notes text area
- Filter by: status, priority, type
- Sort by: priority, spend impact, date created

## Navigation Tabs
```
Overview | Campaigns | Keywords | Search Terms | Proposals
```
